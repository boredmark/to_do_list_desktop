# To do app

## _Desktop application_

A simple, ToDo app built with Python, Tkinter and Sqlite3.
Minimal features, just enough to be useful.
All your need is python 3.6 and above



### Explore

Clone the repo:

```sh
git clone
```

Go to the project directory:

```sh
cd to_do_list
```

Run app:

```sh
python3 main.py
```
